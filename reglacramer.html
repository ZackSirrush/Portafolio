Aplicación móvil: Regla de Cramer

Este proyecto consistió en el desarrollo de una aplicación móvil multiplataforma utilizando Xamarin Forms que permita al usuario capturar los coeficientes y términos independientes de dos ecuaciones lineales con dos incógnitas (X y Y), para resolver una ecuación con los pasos definidos para dicha regla; una vez que se calculan, la aplicación muestra en la página siguiente el resultado obtenido; adicionalmente, el sistema en cuentión contiene parámetros para el control de errores que prevenienen el ingreso de información incorrecta o resultados no definidos

Objetivo: Calcular la regla de Cramer a través del ingreso de coeficientes de dos ecuaciones lineales independientes con dos incógnitas.

Características destacadas: Navegación sencilla de solo dos "activities" (o formularios) donde se muestran: la página principal donde se ingresarán los datos de las ecuaciones y un botón para calcular el resultado, así como un segundo formulario donde se mostrarán los resultados con los pasos para determinarlo.

Tecnologías utilizadas: C# y Xamarin Forms
