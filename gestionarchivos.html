Aplicación de Escritorio: Gestión de archivos TXT

La aplicación de Gestión de Archivos es una herramienta desarrollada en Java que permite a los usuarios crear, abrir, editar, renombrar y eliminar archivos de texto en el sistema de archivos del usuario. Esta aplicación ofrece una interfaz gráfica intuitiva que facilita la manipulación de archivos de texto de manera eficiente.

Objetivo: Facilitar la gestión de archivos de texto en el sistema de archivos del usuario, proporcionando una interfaz gráfica fácil de usar para realizar operaciones comunes como crear, abrir, editar y eliminar archivos.

Características destacadas: Creación de nuevos archivos de texto especificando nombre y contenido, apertura de archivos existentes para visualizar y editar su contenido, renombrado de archivos de texto existentes, eliminación segura de archivos de texto, interfaz gráfica intuitiva y fácil de usar

Tecnologías utilizadas: Java, Swing (para la interfaz gráfica de usuario) y JFileChooser
