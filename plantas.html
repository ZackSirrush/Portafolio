Aplicación de Escritorio: Plantas endémicas

La aplicación en cuestión tiene como propósito optimizar el manejo de una base de datos remota con más de 87,000 registros de ventas; para ello, el sistema ejecuta sentencias SQL que llaman a una base de datos remota y otra local, mostrando el resumen de ambas y permitiendo el filtrado de información de acuerdo con los datos ingresados por el usuario, así mismo, los usuarios podrán registrar ventas, editar y eliminar; el programa en cuestión incluye manejo de excepciones que previenen el cierre inesperado de la aplicación.

Objetivo: Optimizar una base de datos remota, ejecutar consultas mediante MySQL y administrar el sistema de ventas de una empresa dedicada a la venta de plantas endémicas.

Características destacadas: Diseño simple, navegación intuitiva, conexión a BD mediante MySQL, ejecución de consultas SQL

Tecnologías utilizadas: C#, .NET, Formularios (Windows Forms) y MySQL para el manejo de la base de datos local y remota
