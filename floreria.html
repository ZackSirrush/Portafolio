Aplicación de Escritorio: Florería para toda ocasión

Este proyecto consistió en el desarrollo de una aplicación de escritorio para una florería, que permitiera gestionar reportes de empleados y pedidos. La aplicación facilita la generación, filtrado y exportación de reportes en formato PDF, así como la visualización de pedidos realizados por cada empleado. Además, ofrece la posibilidad de enviar reportes por correo electrónico al administrador y de guardarlos en la base de datos.

Objetivo: Proporcionar una herramienta eficiente para la gestión de reportes de empleados y pedidos en una florería, permitiendo listar actividades y pedidos ingresados con el propósito de generar informes detallados de las actividades realizadas y pendientes para la toma de decisiones.

Características destacadas: Generación de reportes de empleados y pedidos, filtrado de reportes por fechas y selección de empleado, exportación de reportes en formato PDF, visualización de pedidos realizados por cada empleado, envío de reportes por correo electrónico al administrador, almacenamiento de reportes en la base de datos, chat interno mediante la aplicación, uso de hilos para cronometrar tiempos de trabajo, conexión a base de datos.

Tecnologías utilizadas: Java, JDBC (Java Database Connectivity), JasperReports, iReports, Swing (para la interfaz gráfica de usuario) y MySQL (como base de datos relacional)
